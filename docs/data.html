Missing Data And Imputation

Chapter 2 Data

The dataset that we used was found on kaggle and came from Glassdoor. This dataset focuses on compensation and has 9 variables. This data was constructed in efforts to aid testing the pay gap between men and women.

2.1 Variables

The variables in the dataset were:

Job Title - which is a categorical variable consisting of Financial Analyst, Graphic Designer, Software Engineer, IT Warehouse Associate, Driver, Sales Associate, Data Scientist, Marketing Associate, and Manger

Gender - which is a categorical variable consisting of Male and Female

Age - which is a continuious variable ranging between 18 and 65 years old.

Performance Evaluation - which is a categorical variable with ranked values 1,2,3,4,5 based on perfomance. 1 being the lowest and 5 being the highest.

Education - which is a categorical variable consisting of High School, College, Masters, PhD.

Department - which is a categorical variable consisting of Operations, Sales, Management, Administration, Engineering

Senority - which is a categorical variable with ranked values 1,2,3,4,5 based on perfomance. 1 being the lowest and 5 being the highest.

Base Pay - which is a continuious variable ranging between 34.2k - 180k

Bonus - which is a continuious variable ranging between 1.7k - 11.3k
